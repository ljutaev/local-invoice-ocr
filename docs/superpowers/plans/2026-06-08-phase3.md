### Task 1: schema — Invoice.exported_at + migration
- `models.py`: add to `Invoice`: `exported_at: Mapped[datetime | None] = mapped_column(DateTime, nullable=True)`
- `db.py`: after `create_all`, add idempotent migration:
```python
from sqlalchemy import inspect, text
insp = inspect(_engine)
cols = [c["name"] for c in insp.get_columns("invoices")]
if "exported_at" not in cols:
    with _engine.begin() as conn:
        conn.execute(text("ALTER TABLE invoices ADD COLUMN exported_at DATETIME"))
```
- Commit.
